# Caramelo POS - Motorcycle Dealership System

ระบบจัดการร้านจำหน่ายมอเตอร์ไซค์แบบครบวงจร

## Features
- 🏍️ ระบบจัดการสินค้าคงคลัง (Inventory Management)
- 👥 ระบบจัดการลูกค้า (Customer Management)  
- 💰 ระบบขาย POS (Point of Sale)
- 📦 ระบบคลังสินค้า (Storage Management)
- 🎁 ระบบของแถม (Gift Management)
- 📊 รายงานและวิเคราะห์ (Reports & Analytics)
- 📋 ติดตามการจดทะเบียน (Registration Tracking)
- 💳 ระบบชำระเงิน NPG (NPG Payments)
- 🔐 JWT Authentication

## Tech Stack
- **Frontend:** Next.js 14.1.3, TypeScript, Tailwind CSS
- **Backend:** Django 5.0.2 REST API
- **Authentication:** JWT with role-based access
- **Compliance:** PDPA compliance
